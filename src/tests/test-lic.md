# `test-lic.cpp` 作用及執行流程

`test-lic.cpp` 是一個用於測試 `qcap2` 軟體授權機制客戶端的簡單應用程式。

## 執行流程

1.  **授權內容宣告**：程式使用 `QCAP2_DECL_LIC` 巨集宣告一個授權內容（license context）變數。此變數用於保存授權檢查的狀態。

2.  **授權註冊**：在測試開始時，程式呼叫 `qcap2_lic_register()` 函式。此函式將授權內容註冊到 `qcap2` 授權系統中，這可能會觸發一個背景程序來驗證授權（例如，透過連接授權伺服器或讀取本機的授權金鑰檔案）。

3.  **狀態檢查**：程式隨後進入一個迴圈，等待使用者輸入。

4.  如果使用者按下 `t` 鍵，程式會呼叫 `QCAP2_LIC_SUCCEEDED()` 巨集來檢查授權的目前狀態。

5.  最後，如果授權有效，它會在主控台上印出 "GRANTED"；如果無效，則印出 "DENIED"。

此程式本身不執行任何實際需要授權的功能。其唯一目的是展示客戶端應用程式如何註冊授權並查詢其驗證狀態。
